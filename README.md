# Edu-Insight: Intelligent Student Performance & Retention Analytics System

## Overview
Edu-Insight is a MERN-stack web application designed for universities to automate attendance tracking and apply rule-based early-warning analytics to identify At-Risk students before detention.

The system focuses on proactive academic intervention through data visualization and structured analytics.

## Project Category
University / Community-Based Project

## Technology Stack
- Frontend: React, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub

## Current Status
Project initialized. Core system architecture and modules are under development.
